# Components Directory

This directory contains all React components organized by feature.

## Structure:

### `/layout` - Layout components
- Navbar, Footer, Layout wrapper

### `/game` - Game-related components
- GameCanvas, FallingObject, ScoreDisplay, GameControls, GameOverModal, GameTutorial

### `/leaderboard` - Leaderboard components
- LeaderboardTable, LeaderboardCard, LeaderboardFilters, PlayerStats

### `/wallet` - Wallet integration components
- ConnectWallet, WalletButton, WalletModal, RegisterModal

### `/ui` - Reusable UI components
- Button, Card, Modal, Loading, Toast

**Important**: Keep each component under 500 lines. Break into smaller components if needed.
