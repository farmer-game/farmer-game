# Hooks Directory

Custom React hooks for the application.

## Hooks:
- `useWallet.ts` - Wallet connection state management
- `useContract.ts` - Smart contract interactions
- `useGame.ts` - Game state management
- `useLeaderboard.ts` - Leaderboard data fetching
- `useChainhook.ts` - Chainhook event subscription (optional Phase 2)

Each hook should be under 200 lines of code.
